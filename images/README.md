# 行程照片資料夾

把照片放進這個資料夾，**用下表指定的檔名**（小寫、`.jpg`），
commit / push 後就會出現在手冊網站上，**手機、電腦雙端都看得到**。

- 格式：`.jpg` 最保險（也支援 `.jpeg`、`.png`）。
- 不必每個都放；沒放的會自動顯示原本的網路圖或墨綠色塊。
- 放好後告訴我，我幫你 `git add images && commit && push`。
- 圖片會以半透明墊在墨綠底下（維持手冊風格），所以選圖不必太講究。
